# users-crud-demo
Prerequisite requirements

  - NodeJs v14
  - MongoDB

Services list

  - frontend
  - backend

How to run frontend:
```sh
$ cd frontend
```
```sh
$ yarn
```
```sh
$ yarn build
```
```sh
$ serve -s build
```
How to run backend:
```sh
$ cd backend
```
```sh
$ yarn
```
```sh
$ yarn start
```
